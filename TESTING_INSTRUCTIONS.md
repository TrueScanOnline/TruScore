## üîß Troubleshooting

### "Can't connect" or "Connection failed"
- **Check:** Developer has Terminal 1 running
- **Wait:** Sometimes takes 30-60 seconds first time
- **Try:** Disconnect and reconnect
- **Check:** You have internet connection

### iPhone: "Open in Expo Go?" doesn't appear
- **Check:** Expo Go is installed from App Store
- **Try:** Open Expo Go app once, then try again
- **Try:** Use Method 2 (paste in Safari) instead

### Android: QR code doesn't scan
- **Try:** Tap "Enter URL manually" instead
- **Try:** Use better lighting
- **Try:** Get closer to QR code

### App won't load
- **Wait:** First load can take 30-60 seconds
- **Check:** Internet connection is working
- **Try:** Close Expo Go and reconnect
