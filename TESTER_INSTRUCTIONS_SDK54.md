### "Can't connect" or "Connection failed"
- **Check:** Developer has Expo running and tunnel active
- **Try:** Ask developer to check Terminal 1 is still running
- **Wait:** Sometimes takes 30-60 seconds

### "Version mismatch" or "SDK version error"
- **Fix:** Update Expo Go app from App Store
- **Or:** Delete and reinstall Expo Go
- **Should be SDK 54** (latest version from App Store)

### Can't find "Enter URL manually"
- **Try:** Tap screen (if you see camera view)
- **Try:** Look for menu icon (â˜°) - usually top left/right
- **Try:** Swipe up from bottom
- **Try:** Look at bottom of screen for text input
- **Or:** Use Safari method (Option C)

### App won't load
- **Wait:** First load can take 30-60 seconds
- **Check:** Internet connection
- **Try:** Disconnect and reconnect
- **Try:** Close and reopen Expo Go
