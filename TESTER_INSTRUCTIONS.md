## ðŸ“± How to Connect to TrueScan App

### Step 1: Install Expo Go (if not already installed)
1. Open App Store on your iPhone
2. Search for "Expo Go"
3. Install the app (free)

### Step 2: Get the Connection URL
You'll receive a URL from the developer that looks like:
```
exp://xxxxx.exp.direct:443
```

### Step 3: Connect in Expo Go

#### Option A: If You See "Enter URL manually" Button
1. Open Expo Go app
2. Tap "Enter URL manually" button
3. Paste the URL you received
4. Tap "Connect"
5. Wait 10-30 seconds
6. **Done!** App loads

#### Option B: If You Don't See "Enter URL manually"
**Try these steps:**
1. Open Expo Go app
2. **Look for menu icon** (â˜°) or settings icon - tap it
3. **Or tap the screen** (if you see camera view) - might show options
4. **Or look at bottom of screen** - might have text input or button
5. **Or try typing** - might reveal the input field
6. Once you find where to enter URL, paste it and connect

#### Option C: Deep Link from Safari (Alternative)
1. Copy the `exp://xxxxx.exp.direct:443` URL
2. Open Safari on your iPhone
3. Paste the URL in the address bar
4. Tap Go
5. Safari should prompt: "Open in Expo Go?"
6. Tap "Open"
7. Wait for app to load

### Step 4: Wait for App to Load
- First load might take 10-30 seconds
- You'll see loading spinner
- Then TrueScan app appears!

## ðŸ”§ Troubleshooting

### "Can't connect" or "Connection failed"
- **Check:** Make sure developer has Expo running
- **Try:** Ask developer to restart Expo tunnel
- **Wait:** Sometimes takes 30-60 seconds to connect

### "Server not found"
- **Check:** Make sure URL is correct (starts with `exp://`)
- **Try:** Ask developer to check if Expo tunnel is still running
- **Check:** Make sure you have internet connection

### Can't find "Enter URL manually"
- **Try:** Look for menu icon (â˜°) - usually top left or top right
- **Try:** Tap anywhere on screen
- **Try:** Swipe up from bottom
- **Try:** Look for "Connect" or "Paste URL" text
- **Try:** Use Safari method (Option C above)

### App won't load after connecting
- **Wait:** Sometimes takes 30-60 seconds on first load
- **Check:** Internet connection
- **Try:** Disconnect and reconnect
- **Try:** Close and reopen Expo Go app
